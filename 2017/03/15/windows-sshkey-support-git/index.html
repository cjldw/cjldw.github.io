windows 生成sshkey 实现github git协议克隆提交代码

Mar 15, 2017

一般的, 我们从github, 或是私有仓库gitlab中克隆代码, 都是使用 git clone https://github.com/u/xxx.git, https协议相比git, (git@github.com:u:/xx.git) 会慢很多, 而且要保存帐号密码到本地,才能实现免帐号密码登入. 使用ssh协议就方便很多了.
